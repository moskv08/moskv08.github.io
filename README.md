# Three.js App - 3D Globe

## Overview

Spinning Globe with Atmosphere in space. Created by use of:

* Vite Framework
* Three.js libray
* Custom _vertex_ & _fragement_ shaders written in GLSL

## Setup

_Clone_ or _Download_ the project and navigate to the project folder.

```
npm install -g vite
npm install

npm run dev
```
